# 🎨 RISE Project Design Assets

This directory contains the architectural and visual planning for the RISE Bookstore Management System.

## 1. Navigational Site Map
Defines the user journey and system hierarchy. The structure ensures a "3-click" maximum depth for core administrative tasks to maximize operational efficiency.

## 2. Low-Fidelity Wireframes
* **Inventory Dashboard:** Optimized for rapid stock audits and data entry.
* **Sales Terminal (POS):** A split-pane design focused on transaction speed and accuracy.
* **Manufacturer Ordering:** A card-based layout for intuitive tracking of logistics and shipments.

## 3. Visual Design System
* **Primary Color:** Midnight Blue (#2C3E50) – Chosen to establish professional trust and visual hierarchy.
* **Success Color:** Emerald Green (#27AE60) – Reserved for growth-oriented actions (Add Title, Complete Sale).
* **Typography:** Focused on readability for high-density data environments.
